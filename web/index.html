seiswg

2-D finite-difference seismic solver — Rust + WebGPU backend, running entirely in-browser via WebAssembly.

All textareas in the simulator are editable. model.js runs in-browser to generate binary model files. Source format: x z type f0 t0 ang amp

Open Simulator Open in GitHub

Overview

A GPU-accelerated re-implementation of the Python/CUDA seispie solver. Every compute kernel runs as a WGSL compute shader via wgpu, targeting Vulkan, Metal, and DX12 from a single code-base. The in-browser build compiles to WebAssembly + WebGPU — no server or install required.

Input file formats

Model files

Binary files named proc000000_<param>.bin in the model directory. Format: 4-byte little-endian int32 giving npt, followed by npt × float32.

FileDescription
proc000000_x.binx-coordinates [m]
proc000000_z.binz-coordinates [m]
proc000000_vp.binP-wave velocity [m/s]
proc000000_vs.binS-wave velocity [m/s]
proc000000_rho.binDensity [kg/m³]
proc000000_lambda/mu/nu.binLamé + micropolar coupling [Pa] (spin only)
proc000000_j.binMicro-inertia [kg/m³] (spin only)

sources.dat

x_m  z_m  type  f0_Hz  t0_s  angle_deg  amplitude
FieldDescription
x_m, z_mSource position in metres
typeUnused — set to 0
f0_HzRicker dominant frequency
t0_sRicker delay time
angle_degForce azimuth (0 = +x, 90 = +z)
amplitudeForce amplitude

stations.dat

x_m  z_m

Output files

FileComponentLayout
uy_NNNNNN.npySH displacement yNumPy [nrec, nt]
ux_NNNNNN.npyPSV displacement xNumPy [nrec, nt]
uz_NNNNNN.npyPSV displacement zNumPy [nrec, nt]
ry_NNNNNN.npySpin total micro-rotation yNumPy [nrec, nt]
yi_NNNNNN.npyIsolated rotationNumPy [nrec, nt]

NNNNNN is the zero-padded source index. Velocity snapshots (save_snapshot > 0) are raw float32 proc{it:06}_v*.bin in output/.

Physics

4th-order staggered-grid finite-difference scheme in space, first-order explicit time stepping:

  • SH — anti-plane shear: vy, σxy, σzy
  • PSV — in-plane: vx, vz, σxx, σzz, σxz
  • Micropolar/spin — classical PSV + micro-rotation φy via couple-stress tensor; yields total rotation, isolated rotation, and curl rotation seismograms

Absorbing boundaries use a Gaussian taper applied multiplicatively to velocity fields after each update.

Building the WASM package

# install wasm-pack (one-time)
cargo install wasm-pack --locked
rustup target add wasm32-unknown-unknown

# compile
wasm-pack build --release --target web --out-dir web/pkg

# serve locally
npx serve web

Then open localhost:3000 in Chrome ≥ 113, Edge ≥ 113, or Safari ≥ 18.
